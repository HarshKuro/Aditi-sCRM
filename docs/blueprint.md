# **App Name**: Prospex CRM

## Core Features:

- Dashboard: Dashboard with key metrics and summaries of customers, tasks, and sales data. This will initially use mock data, with placeholders for connecting to data sources later.
- Customer Management: Customer list with the ability to search and filter customers, view details, and assign employees. Access is role-based: Employees see only their assigned customers. This will use mock data for customer details.
- Task Management: Task list allowing users to add, edit, and view tasks, with filters for status and due date. Priority indicators will use color-coding based on the due date, but these will not initially involve automatic notifications or reminders.
- Sales Pipeline: Sales pipeline visualized as a Kanban board. Users can move sales opportunities between stages, but advanced forecasting is excluded from the MVP.
- Role-based Authentication: Login page with authentication based on roles (Admin, Employee). Mock authentication for MVP.

## Style Guidelines:

- Primary color: Strong, vibrant blue (#4285F4) to convey trust and reliability in a business context.
- Background color: Light, desaturated blue (#E8F0FE) to ensure legibility and create a calming workspace environment.
- Accent color: Soft purple (#A479C9) to highlight interactive elements and provide a modern feel.
- Body and headline font: 'Inter' sans-serif, for a modern, clean, objective feel. This will be used throughout the app, because it is appropriate for both headlines and body.
- Card-based layout with consistent padding and margins.
- Simple, professional icons from a set like 'Boxicons'.
- Subtle animations for transitions and loading states.
